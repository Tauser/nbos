# Biblioteca Exploratoria de Presets (F4)

Status: exploratorio (nao congelado).

Objetivo:
- explorar linguagem visual
- medir legibilidade entre estados
- preparar tuning antes do congelamento oficial da library

Readability tiers:
- Tier A (Immediate): leitura instantanea em poucos frames
- Tier B (Balanced): boa legibilidade com mais nuance
- Tier C (Nuanced): identidade mais rica, exige tuning de composicao

Presets exploratorios atuais:
- clarity_neutral (Tier A): baseline limpa e simetrica
- focused_attend (Tier A): foco alto e abertura reduzida
- calm_low_power (Tier B): energia baixa e foco interno
- curious_observe (Tier B): eixo alto + curiosidade
- playful_lift (Tier C): roundness e diagonal suave
- attentive_lock (Tier C): lock atencional com contraste de foco

Notas arquiteturais:
- presets sao data-driven (sem hacks por preset)
- biblioteca e exploratoria: nao representa congelamento da library final
- composicao governada continua sendo gate para consolidacao futura
